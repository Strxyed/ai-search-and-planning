# AI Search and Automated Planning

This repository contains an Artificial Intelligence coursework project focused on solving search and planning problems using Constraint Satisfaction Problems (CSPs) and Automated Planning.

The project consists of two parts:
- **Sudoku Solver** implemented in Python using CSP techniques.
- **Lunar Rover Planning** implemented using PDDL for automated mission planning.

---

## Features

### Sudoku Solver
- Interactive Sudoku GUI built with Tkinter
- Backtracking search algorithm
- Minimum Remaining Values (MRV) heuristic
- Forward checking
- Puzzle loading from text and CSV files
- Performance metrics (execution time, recursive calls, backtracks)

### Automated Planning
- PDDL domain modelling
- Multiple lunar rover mission scenarios
- AI planning for autonomous task execution
- Extended planning domain implementation
